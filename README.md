## Barebones gulp project

Barebones gulp.js project. Gives you local server, liveReload, Sass, & jshint.

### getting started

Install [node](http://nodejs.org/) & [gulp](http://gulpjs.com/)

`brew install node`, `npm install -g gulp`

Install packages

`npm i`

Run Gulp

`gulp`
